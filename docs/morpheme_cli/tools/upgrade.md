# Upgrade CLI

This command is used to upgrade the **Morpheme CLI** tool itself to the latest version available on pub.dev.

```bash
morpheme upgrade
```

## Options

```bash
morpheme upgrade [options]
```

To see all available options and flags, run `morpheme upgrade --help`.

### Available Options

| Option | Abbr | Description |
|---|---|---|
| `--force` | `-f` | Force upgrade even if already on latest version |
| `--verbose` | `-v` | Show detailed output during upgrade process |

## Examples

**Standard Upgrade:**

```bash
morpheme upgrade
```

**Force Upgrade:**

```bash
morpheme upgrade --force
```
