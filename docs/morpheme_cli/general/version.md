# Version

This command displays the currently installed version of the Morpheme CLI.

```bash
morpheme -v
```

Alternative:

```bash
morpheme --version
```

## Output

```bash
Morpheme CLI 4.0.0
```

## Checking for Updates

To upgrade to the latest version of Morpheme CLI, use:

```bash
morpheme upgrade
```
